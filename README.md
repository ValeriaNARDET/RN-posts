React Native (Expo) App contein:

- TypeScript
- Expo Router
- Redux Toolkit
- Authentication
- CRUD (with Axios)

To clone and set the project:

```bash
git clone <repository-url>
cd <project-folder>
npm install
```

To start the progect

```bash
npx expo start
```

or 

```bash
npx expo start -c
```

and then press 'a' (for android emulator) or press 'i' (for iOS)
